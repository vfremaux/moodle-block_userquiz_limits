This plugin is a control bloc for setting up users attempts max values xwhen
using the question accessrule usernumattemps.

Usernumattempts allow setting individually the max amount of atempts a user can do in a quiz.

This blocs allows administrate those settings for any quiz having this accessrule enabled.
